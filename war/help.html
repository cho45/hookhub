Help

What is hookhub?

hookhub is a hub of web hooks. You can write JavaScript to route web hooks to other web services. There is similar service Scriptlets, too.

You can create web hooks receiver very easy, if you can write programs. You can also "fork" any other code and use it, even if you cannot write programs.

How do I set my icon?

Associate your email address by Gravatar.

External Providing Web Hooks Services

  • Hatena::Bookmark (Social Bookmark Service): Post bookmarked hook, Post friends bookmarked hook, Post stared hook, Etc...
  • Google Code (Repository sharing service): Post commit hook
  • GitHub (Repository sharing service): Post pushed hook
  • BitBucket (Repository sharing service): Post pushed hook
  • PostBin (Debug web hooks)

Code Reference

The code has some restrictions:

  • Done within 5seconds.
  • 1000 recursion.
  • Can't re-invoke same hook in 10seconds.

HTTP Request

http(opts: Object) => Object
HTTP request synchronously-executed. 
// Example:
http({
	method  : "post",
	url     : url,
	data    : http.data(params),
	headers : {
		"Content-Type":"application/x-www-form-urlencoded"
	}
})

// Sample returned value
({
	code: 200,
	headers : { 'Content-type':"text/plain" },
	body : "Hello"
})
http.post(url: String, params: Object) => Object
HTTP request using POST method.
http.get(url: String) => Object
HTTP request using GET method.
http.data(params: Object) => String
Construct post data from Object. 
({
	foo: "bar",
	baz: "piyo"
})
//=>  "foo=bar&baz=piyo"

({
	foo: "bar",
	baz: ["piyo", "puyo"]
})
//=>  "foo=bar&baz=piyo&baz=puyo"

Mail

You can send mail only yourself.

mail(title, body)
Mail to you with title and body. Only send 1 mail in a hook.
